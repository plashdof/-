우분투 업데이트 : sudo apt-get update   
c++ 코딩가능하게 업데이트 : sudo apt-get install build-essential gdb

pwd : 현재 폴더 위치
whoami : 어떤 사용자인지 알려줌
sudo passwd root : 관리자 권한 비번설정
su : 관리자 권한 모드로 전환
exit : 관리자 권한 모드 -> 일반 모드

cd (폴더명) : 해당 폴더로 위치 이동
cd .. : 전단계 폴더로 위치 이동
rm (파일명) : 파일삭제
마우스 우클릭 : 붙여넣기





	<vim 에디터>

	vim -> 메모장과 같은 편집기 에디터
	vim 에디터 접속하기 -> vim 치고 엔터
                               	       vim 파일명(ex) helloworld.cpp)

	:q : vim 모드 빠져나가기
	:wq : 저장후 vim 모드 빠져나가기
	:! : 강제종료

	i : 입력모드( 커서 앞쪽 편집모드 )
	a: 입력모드( 커서 뒤쪽 편집모드 )
	esc : 입력모드 빠져나오기

	:set paste : 윈도우 글 칸 맞춰서 복붙모드.

	V : 줄단위 선택모드
	v : 선택모드
	y : 복사
	p : 붙여넣기
	H : 화면 맨 위로
	L : 화면 맨 아래로

	u : 실행취소
	ctrl + r : 다시 되돌림

	<강제종료 후 attention 에러뜰때 대처법>
	vi -r (파일명)  : 진행중이었던거 확인후 :wq 로 저장
	ls -all : 스왑파일명 정확히 확인
	rm (스왑파일명) : 스왑파일 제거
	그후 다시 vim 에디터로 진행


ls : 현재 폴더 보기
ls -al : 폴더 상세보기
cat 파일명 : vim 에디터 들어가지 않고, 파일 텍스트형식으로 출력

g++ 파일명 -o 실행파일명 : 실행파일 만들기.
./(실행파일명) : 파일 실행시키기
	실행파일명 안붙이고 g++ 파일명 하면, 기본으로 a.out 이라는 파일명으로 생성

mkdir (폴더명) : 리눅스에 새 폴더 만들기

	리눅스 에서 파일생성? -> vim 에디터 이용! 
             	                 -> vim 파일명 하고 에디터 들어간후 (메모장이면 .txt 소스파일이면 .cpp)
                         		    -> 내용작성후 :wq
